# SJ-Library
Anndroid Studio library

## Download

Add it in your root build.gradle at the end of repositories:
```
allprojects {
		repositories {
			...
			maven { url 'https://jitpack.io' }
		}
	}
```

Add implementation:
```
dependencies {
	        implementation 'com.github.slavce14:SJ-Library:1.3.0.1'
	}
```
